# Sistema de Autoservicio - Proyecto Programación III

Este proyecto es una plataforma integral desarrollada en equipo para la materia Programación III. La solución centraliza toda la gestión de un sistema de autoservicio en un único repositorio estructurado de forma modular.

## 🚀 Estructura del Proyecto

El repositorio está organizado en tres módulos independientes que trabajan en paralelo:
* **`Parte-front`**: Interfaz de usuario principal para el cliente del autoservicio.
* **`parte-back`**: Servidor y API central para el manejo de datos, lógica de negocio y base de datos.
* **`parte_dashboard`**: Panel de administración interno para el control de stock, reportes y gestión del sistema.

## 🛠️ Tecnologías Utilizadas

* **Frontend & Dashboard:** JavaScript (puedes agregar acá si usan React, HTML/CSS, etc.)
* **Backend:** Node.js / Express (o las tecnologías que estén usando para el servidor)
* **Control de Versiones:** Git y GitHub para el flujo de trabajo colaborativo.
